### 📊 Demand Forecasting & Inventory Optimization
**Machine Learning · Time Series · Analytics**

</div>

<br/>

An interactive analytics engine built on **1.7M+ real retail sales records** across 54 stores and 33 product families. Compares 5 forecasting methods head-to-head and calculates optimal inventory parameters.

<br/>

<div align="center">

| Feature | Detail |
|:--------|:-------|
| 📈 Forecasting | Moving Avg, Holt-Winters, ARIMA, Prophet, XGBoost |
| 🎯 Accuracy | 8-12% MAPE with automated best-model selection |
| 📦 Inventory | EOQ, Safety Stock, Reorder Point calculator |
| 🏷️ ABC | Pareto classification of 33 product families |
| 🔮 Simulator | What-If analysis across lead times & service levels |
| 📉 Cost Curves | Visual EOQ trade-off analysis |
| 🔍 EDA | Seasonality, promo impact, day-of-week patterns |
| ⚡ Interactive | Filters by product, store, time aggregation |

</div>

<br/>

```
Tech: Python · Streamlit · statsmodels · scikit-learn · XGBoost · Plotly
Data: 3M+ records · 54 stores · 33 families · 4.5 years
```
